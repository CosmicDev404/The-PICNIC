An interactive horror web based CYOA game. The picnic is more than yet another indie web game, it is intended to be a foundation for new curious creations. In a CYOA style storytelling, this game guides the player through the horrors of a lucid nightmare. Based in the hilly terrain of Shiwaliks in India, player encounters various dilematic situations and psychological horror elements based only on the auditory experience.

## Instructions

- Download all the files in same folder/directory
- Run _"index.html"_ file for main game on any web browser locally
- Make sure sound is at maximum volume or headphones are used for best experience
- The game can also be played via the official site
- This project is intended for creative usage. It is recommended to use the project as a template for other creative implementations
